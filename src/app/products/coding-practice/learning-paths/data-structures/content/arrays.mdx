# Arrays in Data Structures

Arrays are one of the most fundamental data structures in computer science. They store elements in contiguous memory locations, providing fast access to elements using indices.

## Introduction to Arrays

An array is a collection of similar data elements stored at contiguous memory locations. It is the simplest data structure where each data element can be accessed directly by only using its index number.

### Key Features

1. **Fixed Size**: Arrays have a fixed size determined at creation time
2. **Contiguous Memory**: Elements are stored in consecutive memory locations
3. **Random Access**: Any element can be accessed in O(1) time using its index
4. **Homogeneous Elements**: All elements must be of the same data type

## Basic Operations

### 1. Insertion
- Time Complexity: O(n) for arbitrary position
- Time Complexity: O(1) for insertion at end with space available

```javascript
function insertAt(arr, pos, element) {
  for(let i = arr.length; i > pos; i--) {
    arr[i] = arr[i-1];
  }
  arr[pos] = element;
}
```

### 2. Deletion
- Time Complexity: O(n) for arbitrary position
- Time Complexity: O(1) for deletion at end

```javascript
function deleteAt(arr, pos) {
  for(let i = pos; i < arr.length - 1; i++) {
    arr[i] = arr[i+1];
  }
  arr.length--;
}
```

### 3. Traversal
- Time Complexity: O(n)

```javascript
function traverse(arr) {
  for(let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
  }
}
```

## Common Array Problems

### 1. Array Reversal
```javascript
function reverseArray(arr) {
  let start = 0;
  let end = arr.length - 1;
  while(start < end) {
    // Swap elements
    [arr[start], arr[end]] = [arr[end], arr[start]];
    start++;
    end--;
  }
  return arr;
}
```

### 2. Finding Maximum Element
```javascript
function findMax(arr) {
  let max = arr[0];
  for(let i = 1; i < arr.length; i++) {
    if(arr[i] > max) {
      max = arr[i];
    }
  }
  return max;
}
```

     let first = arr[0], second = -Infinity;
     for(let i = 1; i < arr.length; i++) {
       if(arr[i] > first) {
         second = first;
         first = arr[i];
       } else if(arr[i] > second && arr[i] < first) {
         second = arr[i];
       }
     }
     return second;
   }
